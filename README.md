# 🐾 Encontre.me - Sistema de Animais Perdidos e Achados

Sistema completo para cadastro e gerenciamento de animais perdidos e achados, com suporte a upload de fotos.

## 🚀 Tecnologias

- **Frontend:** HTML5, CSS3, Vue.js 2
- **Backend:** Node.js, Express
- **Banco de Dados:** MySQL
- **Upload de Fotos:** Multer
- **Controle de Versão:** Git

## 📦 Como Rodar o Projeto

### Pré-requisitos

1. **Node.js** (versão 14 ou superior)
2. **MySQL** (versão 8 ou superior)
3. **Git** (opcional, para clonar)

1. **Clone o repositório**
